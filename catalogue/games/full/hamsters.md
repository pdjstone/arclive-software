# Hamsters

## Description

Cartoon 2D platformer where you play a small furry creature with a large mallet which you must use to squish cute animals.

## Quick Start
Controls:
- <kbd>Z</kbd> - Left
- <kbd>X</kbd> - Right
- <kbd>Enter</kbd> - Use hammer/jump (hold down longer for bigger swing)
